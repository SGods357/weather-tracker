# weather-tracker
Intro to Data Science Assigment for tracking Fiordland Naitonal Park's weather
It is an agent that fetches the weather data for a specific location in the world, specifically the Fiordland Naitonal Park. I built this project to help decide whether Fiordland National Park would be a good place to camp based on recent and upcoming temperatures. It uses Open-Meteo API data, including historical daily high/low temperatures and a 7-day forecast, then saves the results as CSV files.
